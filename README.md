# 盛土法面安定計算 Webアプリ

道路土工の実務フローを意識した、依存ライブラリ不要の Web アプリです。

- 無限長斜面法（浅層すべりの予備検討）
- Fellenius法（円弧探索付き分割法による概略照査）

## 使い方

```bash
python -m http.server 8000
```

ブラウザで `http://127.0.0.1:8000/web/` を開き、入力値を設定して計算してください。

## テスト

```bash
python -m unittest discover -s tests -p 'test_*.py'
```

## 道路土工指針への対応方針

このアプリは、道路土工指針の考え方に沿って「すべり形態に応じた計算手法を選ぶ」ための**予備検討**を目的としています。

- 浅層・平面すべりに対しては無限長斜面法
- 円弧すべりに対しては、法肩〜法尻を通る候補円弧を探索して最小安全率を採用するFellenius法

> 注意: 本アプリは教育・概略検討用途です。実施設計では、道路土工指針の最新版、地盤調査結果、地震時条件、排水工・抑止工を含む総合照査を必ず実施してください。
