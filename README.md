# 盛土法面の安定計算 WEBアプリ

シンプルな静的HTML/JavaScriptで作成した、盛土法面の概略安全率計算アプリです。

## 概要
- 無限長法面モデルを採用
- 入力: 法面高さ、勾配、γ、c'、φ'、r_u
- 出力: 法面角β、安全率Fs、簡易判定

## 計算式

\[
Fs = \frac{c'}{\gamma H \sin\beta \cos\beta} + \frac{(1-r_u)\tan\phi'}{\tan\beta}
\]

## 使い方
1. `index.html` をブラウザで開く
2. 各パラメータを入力
3. 「計算する」をクリック

## 注意
- 概略検討用です。実務では円弧すべり法（Bishop, Janbu等）や浸透流解析、地震時照査を行ってください。
